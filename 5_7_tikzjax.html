5.7 CONTINUITY OF A FUNCTION AT A POINT

In Sec. 5.6, we have discussed the concept of L.H.L. and R.H.L. Adding one more-step, we can define continuity at a point.

Definition
Continuity at a Point

A function f(x) is said to be continuous at x = a if $$ \lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x) = f(a) $$ i.e. for continuity at a point x = a, we must have i.e. L.H.L. at x=a = R.H.L. at x=a = value of the function at x = a

Diagrammatically, continuity at x = a means graph of the function f(x) from a value slightly less than ‘a’ to a value slightly greater than ‘a’ has no gap, i.e. if we draw the graph with pencil then we don’t have to pick up the pencil as we cross the point where x = a.

Consider the function f: R → R defined by $$ f(x) = \begin{cases} 1, & x \ge 1 \\ 2, & x < 1 \end{cases} $$ See the graph of this function in Fig. 5.7. We note that, if we draw the graph of this function with pencil, then we will have to pick up the pencil as we cross the point where x = 1. Therefore this function is not continuous at x = 1. Also this function is continuous at all points of its domain except at x = 1.

Example 9 Discussing Continuity at a Point
Given:
Discuss the continuity of the following functions at the given point: (i) f(x) = |x| at x = 0 (ii) f(x) = |x - 3| at x = 3 (iii) f(x) = $$ \begin{cases} x^2+1, & x \le 1 \\ 1-x^2, & x > 1 \end{cases} $$ at x = 1 (iv) f(x) = $$ \begin{cases} \frac{x-4}{x-4}, & x \ne 4 \\ 0, & x=4 \end{cases} $$ at x = 4 (v) f(x) = x/|x| at x = 0
StepWorkingResultNote
(i) f(x) = |x| at x = 0 L.H.L. = 0, R.H.L. = 0. Also, f(0) = |0| = 0. Since L.H.L. = R.H.L. = f(0), f(x) is continuous at x = 0. Limits cited from Example 7 of the source unit.
(ii) f(x) = |x - 3| at x = 3 L.H.L. = 0, R.H.L. = 0. Also, f(3) = |3 - 3| = 0. Since L.H.L. = R.H.L. = f(3), f(x) is continuous at x = 3. Limits cited from Example 7 of the source unit.
(iii) Piecewise function at x = 1 L.H.L. = 2, but R.H.L. = 0. As L.H.L. ≠ R.H.L., f is not continuous at x = 1. Limits cited from Example 7 of the source unit.
(iv) Piecewise function at x = 4 The source states L.H.L. = 1 but R.H.L. = 1. It then concludes L.H.L. ≠ R.H.L. Therefore, f(x) is not continuous at x = 4. There is a logical contradiction in the source's reasoning (1=1, but it states they are not equal). The correct reasoning is that LHL=RHL=1, but f(4)=0. Since the limit (1) does not equal the function value (0), it is not continuous.
(v) f(x) = x/|x| at x = 0 L.H.L. = lim_{x->0^-} x/(-x) = -1. R.H.L. = lim_{x->0^+} x/x = 1. Since L.H.L. ≠ R.H.L., the limit does not exist. Hence f is not continuous at x = 0.
Example 10 Finding Constants for Continuity
Given:
The function f is continuous at x = 2: $$ f(x) = \begin{cases} 7, & x < 2 \\ a+5, & x = 2 \\ ax+b, & x > 2 \end{cases} $$
Find: The values of a and b.
StepWorkingResultNote
Calculate L.H.L. at x=2 L.H.L. = lim_{x->2^-} f(x) = lim_{x->2^-} 7 7
Calculate R.H.L. at x=2 R.H.L. = lim_{x->2^+} f(x) = lim_{x->2^+} (ax+b) 2a + b
Find the function value at x=2 f(2) a + 5
Apply continuity condition For f to be continuous at x=2, we must have L.H.L. = R.H.L. = f(2). 7 = 2a + b = a + 5
Solve for a From 7 = a + 5 a = 2
Solve for b From 7 = 2a + b, substitute a=2. 7 = 2(2) + b = 4 + b. b = 3
Final Answer a = 2, b = 3
